# Memory-Management-Chatbot

## Dependencies for Running Locally

- cmake >= 3.11
- make >= 4.1 (Linux, Mac), 3.81 (Windows)
- gcc/g++ >= 5.4
- wxWidgets >= 3.0

## Build Instructions

- Clone this repo
- Make a build directory in the top level directory: `mkdir build && cd build`
- Compile: `cmake .. && make`
- Run: `./membot`
